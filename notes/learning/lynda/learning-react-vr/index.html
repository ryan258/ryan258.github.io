Initialization of a Project

  1. Install nodejs to get npm
  2. Get the CLI
    $ npm install -g react-vr-cli
  3. Create a new project
    $ react-vr init [project name]
  4. $ cd [project_name]
  5. yarn start

3D Coordinates and Transforms

3d Axis

  • OpenGL 3D Coordinate System
  • Any component can be placed in 3D space
  • Y axis points p, not down
  • Use meters for distance and dimensions
  • use degrees for rotation

Transforms - position items in the code

  • Extends the React transforms to be 3d
  • Array of objects within a style
  • Each object - command, properties
  • Examp - scale 0.5
