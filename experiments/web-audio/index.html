Making Music
w/ the Web Audio API

Steps:

  1. Clone this Repo
  2. Pick a song and find a MIDI file
  3. Run that MIDI file through ToneJS's converter
  4. Create a new folder and setup a synth
  5. Paste output of MIDI conversion
  6. Clean it up
  7. Preview
